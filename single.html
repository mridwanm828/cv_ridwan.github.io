Tips and Tricks on How to Learn Programming for Beginners Using YouTube Tutorial

Using YouTube tutorials is one of the most popular ways to learn programming for beginners. Here are some tips and tricks that can help you in the learning process using YouTube tutorials:

#1. Set Goals

Set your study goals before starting. Do you want to learn a specific programming language, understand basic programming concepts, or create a specific project? Setting clear goals will help you focus on finding relevant tutorials.

#2. Look for Trusted Channels

Do some research to find trusted and quality YouTube channels on the programming topic you want to study. Review reviews and comments from other users to ensure the quality of the content. Some popular programming channels are "Traversy Media", "freeCodeCamp", "The Net Ninja" and "Codecourse". And my vocational high school teacher recommended "Web Programming UNPAS" and "Dea Afrizal".

Web Programming UNPAS Dea Afrizal

#3. Develop a Lesson Plan

Creating a lesson plan is an important step. Start with videos that teach basic concepts and move on to more complex topics. Make sure you understand and apply the concepts before moving on to the next topic.

#4. Practice Actively

Programming is a hands-on skill, so it's important to take the time to practice. After watching the tutorial, try applying the concepts and code you've learned by creating small projects or exercises.

#5. Read Documentation

Apart from watching tutorials, it is also important to read the official documentation of the programming language you are learning. The documentation will provide more detailed information about the syntax, functions, and features of the language.

#6. Take on Practical Projects

Trying out real projects is a great way to practice your programming skills. Follow tutorials that provide projects that you can follow step by step. In this way, you will gain practical experience in creating a working app or website.

#7. Don't Forget the "Learning By Practice" Principle

VLearning to program takes time and consistent practice. Don't focus too much on watching tutorials without immersing yourself in the coding process. The principle of "learning by practice" is very important, so take the time to create your own projects and solve programming challenges.

#8. Build Community

Join online programming communities, such as discussion forums, Facebook groups, or Slack, which can help you interact with fellow beginners or more experienced programmers. Ask your questions and state the problems you're having. Sharing knowledge with others will deepen your understanding.

#9. Evaluate Your Progress

Do periodic evaluations to see how far you are progressing. Repeat previous exercises or projects to ensure solid understanding.
